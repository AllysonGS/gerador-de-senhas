Gerador de Senhas

Um projeto simples de **gerador de senhas** criado como primeiro site funcional. O usuário pode gerar senhas aleatórias de forma prática, com letras maiúsculas, minúsculas, números e símbolos.

---

## Funcionalidades

- Geração de senhas aleatórias com 12 caracteres por padrão.
- Botão para gerar a senha.
- Interface simples e responsiva.

---

## Tecnologias Utilizadas

- **HTML5** – Estrutura do site.
- **CSS3** – Estilização básica.
- **JavaScript** – Lógica de geração de senhas.
